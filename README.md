# TaipeiZoo
台北市立動物園-各動物館區簡介、植物資料

## 簡介
串接台北市政府開放資料平台台北市立動物園API，專案使用MVP+RXJava+Retrofit
* 查看所有動物園區、該園區植物資料
* 查看所有植物資料
